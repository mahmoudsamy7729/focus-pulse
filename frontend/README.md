# Frontend Skeleton

This directory reserves the future Next.js App Router dashboard layout.
Phase 0 keeps it as documentation and placeholders only.

Future frontend work should keep route composition in `app/`, feature-specific
logic in `features/`, reusable UI in `components/`, API client code in
`lib/api/`, shared hooks in `hooks/`, global client state in `stores/`, and
shared TypeScript types in `types/`.
